# dotfiles

- Just my dotfiles with my personal preference on setting up Arch Linux with all
  the apps set up to my taste
- Copied stuff from here and there

## Requirements

- Emacs (29 or higher)
- git
- tmux
- NVIM
- wezterm
- zathura
- zathura-pdf-mupdf
- zsh

## Quick Start

```console
git clone git@github.com:riz-ahmed/.dotfiles.git ./
```
